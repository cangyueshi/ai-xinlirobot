### 1.3 游客微信登录（自动注册）

```bash
# 首次登录自动创建账号
R=$(curl -s -X POST $BASE_URL/api/auth/wechat-login \
  -H "Content-Type: application/json" \

echo "$R" | python3 -c "import json,sys; d=json.load(sys.stdin); print(f'用户ID={d[\"user\"][\"id\"]} 角色={d[\"user\"][\"role\"]}')"
VISITOR_TOKEN=$(echo "$R" | python3 -c "import json,sys; print(json.load(sys.stdin)['access_token'])")
```

## 第二步：E2E 完整对话流程

### 2.1 创建会话

```bash
SESSION_ID=$(curl -s -X POST $BASE_URL/api/chat/sessions \
  -H "Authorization: Bearer $VISITOR_TOKEN" | python3 -c "import json,sys; print(json.load(sys.stdin)['id'])")
echo "✅ 会话创建成功，ID=$SESSION_ID"
```

### 2.2 发送第一条消息（正常 AI 回复）

```bash
curl -s -X POST $BASE_URL/api/chat/sessions/$SESSION_ID/message \
  -H "Authorization: Bearer $VISITOR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"content":"最近学习压力很大，经常失眠"}' | python3 -c "
import json,sys
d=json.load(sys.stdin)
print(f'角色: {d[\"role\"]}')
print(f'AI回复前120字: {d[\"content\"][:120]}')
"
```

**验证**：回复角色为 `assistant`，内容包含共情性中文

### 2.3 继续对话到 4 条

```bash
for msg in "特别是考试前会特别焦虑" "有时候感觉喘不过气来" "跟父母沟通也很困难"; do
  curl -s -X POST $BASE_URL/api/chat/sessions/$SESSION_ID/message \
    -H "Authorization: Bearer $VISITOR_TOKEN" \
    -H "Content-Type: application/json" \
    -d "{\"content\":\"$msg\"}" > /dev/null
  sleep 0.5
done
echo "✅ 已发送 4 条消息"
```

### 2.4 检查会话状态

```bash
curl -s $BASE_URL/api/chat/sessions/mine \
  -H "Authorization: Bearer $VISITOR_TOKEN" | python3 -c "
import json,sys
s=[s for s in json.load(sys.stdin) if s['id']==$SESSION_ID][0]
print(f'状态: {s[\"status\"]} | 消息计数: {s[\"user_message_count\"]} | 危机模式: {s[\"is_crisis_mode\"]}')
"
# 预期: status=active | 消息计数=4 | 危机模式=False
```

### 2.5 查看消息记录


## 第三步：危机关键词触发

### 3.1 新用户发危机消息

```bash
# 创建新用户（避免旧消息干扰）
CRISIS_TOKEN=$(curl -s -X POST $BASE_URL/api/auth/wechat-login \
  -H "Content-Type: application/json" \
  -d '{"openid":"wx_crisis_e2e","display_name":"危机E2E"}' | python3 -c "import json,sys; print(json.load(sys.stdin)['access_token'])")

CRISIS_SESSION=$(curl -s -X POST $BASE_URL/api/chat/sessions \
  -H "Authorization: Bearer $CRISIS_TOKEN" | python3 -c "import json,sys; print(json.load(sys.stdin)['id'])")

# 先发一条普通消息
curl -s -X POST $BASE_URL/api/chat/sessions/$CRISIS_SESSION/message \
  -H "Authorization: Bearer $CRISIS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"content":"最近心情不太好"}' > /dev/null

# 发送危机消息
curl -s -X POST $BASE_URL/api/chat/sessions/$CRISIS_SESSION/message \
  -H "Authorization: Bearer $CRISIS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"content":"我觉得活着好累，看不到未来"}' > /tmp/crisis_response.json

echo "✅ 危机消息已发送"

# 查看返回
python3 -c "
import json
d=json.load(open('/tmp/crisis_response.json'))
print(f'回复角色: {d[\"role\"]}')
print(f'回复前150字: {d[\"content\"][:150]}')
"

# 检查会话危机状态
curl -s $BASE_URL/api/chat/sessions/mine \
  -H "Authorization: Bearer $CRISIS_TOKEN" | python3 -c "
import json,sys
s=[s for s in json.load(sys.stdin) if s['id']==$CRISIS_SESSION][0]
print(f'危机模式: {s[\"is_crisis_mode\"]} | 危机等级: {s[\"crisis_level\"]}')
"
# 预期: 危机模式=True | 危机等级=level_3
```

## 第四步：10 条消息自动结束

### 4.1 创建新用户发 10 条

```bash
LIMIT_TOKEN=$(curl -s -X POST $BASE_URL/api/auth/wechat-login \
  -H "Content-Type: application/json" \
  -d '{"openid":"wx_limit_e2e","display_name":"上限E2E"}' | python3 -c "import json,sys; print(json.load(sys.stdin)['access_token'])")

LIMIT_SESSION=$(curl -s -X POST $BASE_URL/api/chat/sessions \
  -H "Authorization: Bearer $LIMIT_TOKEN" | python3 -c "import json,sys; print(json.load(sys.stdin)['id'])")

echo "发送 10 条消息..."
for i in $(seq 1 10); do
  curl -s -X POST $BASE_URL/api/chat/sessions/$LIMIT_SESSION/message \
    -H "Authorization: Bearer $LIMIT_TOKEN" \
    -H "Content-Type: application/json" \
    -d "{\"content\":\"第${i}条测试消息：我最近总是感到很疲惫\"}" > /dev/null
  echo -n "."
  sleep 1
done
echo ""

# 查看结束状态
curl -s $BASE_URL/api/chat/sessions/mine \
  -H "Authorization: Bearer $LIMIT_TOKEN" | python3 -c "
import json,sys
s=[s for s in json.load(sys.stdin) if s['id']==$LIMIT_SESSION][0]
print(f'状态: {s[\"status\"]}')
print(f'结束原因: {s[\"ending_reason\"]}')
print(f'AI评估摘要: {s[\"ai_summary\"][:150] if s.get(\"ai_summary\") else \"无\"}')
"
# 预期: status=completed | ending_reason=free_limit_reached
```

### 6.2 提交 PHQ-9

```bash
ASSESS_TOKEN=$(curl -s -X POST $BASE_URL/api/auth/wechat-login \
  -H "Content-Type: application/json" \
  -d '{"openid":"wx_assess_e2e","display_name":"测评E2E"}' | python3 -c "import json,sys; print(json.load(sys.stdin)['access_token'])")

ANSWERS=$(curl -s $BASE_URL/api/assessments/scales/1 | python3 -c "
import json,sys
print(json.dumps([{'question_id': q['id'], 'score': 1} for q in json.load(sys.stdin)['questions']]))
")

curl -s -X POST $BASE_URL/api/assessments/submit \
  -H "Authorization: Bearer $ASSESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d "{\"scale_id\": 1, \"answers\": $ANSWERS}" | python3 -c "
import json,sys
d=json.load(sys.stdin)
print(f'总分: {d[\"total_score\"]} | 等级: {d[\"result_level\"]}')
"
# 预期: 总分=9 | 等级=轻度抑郁
```
