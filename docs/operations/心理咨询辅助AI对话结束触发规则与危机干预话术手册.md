# 心理咨询辅助 AI 对话结束触发规则与危机干预话术手册

**版本**：V1.0  
**生效日期**：2026 年 5 月 17 日  
**适用范围**：微信小程序心理咨询辅助 AI 聊天机器人  
**核心原则**：伦理优先 > 安全第一 > 用户体验 > 成本控制

---

## 一、总则

### 1.1 规则优先级

```
危机干预规则 > 智能提前结束规则 > 基础固定条数结束规则
```

- **任何情况下**，只要检测到危机信号，立即暂停所有其他规则，进入危机干预流程
- 未检测到危机时，优先执行智能提前结束规则
- 未触发智能提前结束时，执行基础固定条数结束规则

### 1.2 基本定义

| 术语 | 定义 |
|------|------|
| **有效对话** | 用户发送的与心理困扰、情绪状态、生活事件相关的消息 |
| **无效对话** | 与心理服务无关的闲聊、测试、娱乐性内容 |
| **对话次数计数** | 仅统计用户发送的消息条数，AI 回复不计数 |
| **值班咨询师** | 平台在线的、具备危机干预资质的专业心理咨询师 |

## 二、分级结束对话触发规则

### 2.1 第一级：基础免费评估包正常结束（10 条）

**触发条件**

- 用户已发送满 10 条有效消息
- 未触发任何智能提前结束条件
- 未检测到任何危机信号

**触发时机**

用户发送第 10 条消息，AI 完成回复后自动触发。

**标准过渡话术**

```
感谢你愿意信任我，和我分享这么多心事。

基于我们的交流，我初步判断你目前正经历：
[AI自动生成1-2句话的个性化评估，如：中度的职场焦虑情绪，主要来自近期的项目压力和同事关系，已经开始影响你的睡眠质量]

作为AI助手，我只能为你提供初步的情绪疏导和通用建议。为了更专业、更安全、更有针对性地帮助你解决问题，我为你准备了两个选择：

1️⃣ 预约专业心理咨询师：获得1对1的深度咨询服务，制定个性化解决方案
2️⃣ 购买AI深度倾诉包：9.9元获得30条额外对话，我会陪你继续梳理情绪、探索问题根源

无论你选择哪种方式，我都会在这里支持你。如果你现在需要安静一下，也可以随时回来找我。
```

**后续动作**

- 展示付费 / 转人工入口
- 保存本次对话记录，供后续人工咨询师查看
- 24 小时内不自动发送任何消息打扰用户

---

### 2.2 第二级：AI 智能提前结束（1-9 条内触发）

#### 2.2.1 触发条件 A：核心评估信息已收集完整

**判断标准**（同时满足）：

- 用户清晰说明了主要困扰（主诉）
- 用户说明了困扰的持续时间
- 用户说明了困扰对生活 / 工作 / 学习的影响程度
- 用户没有提出新的问题或需要进一步探讨的内容
- AI 已给出初步评估和 3 条以上可执行的建议

**触发时机**：AI 完成建议回复后自动触发

**标准话术**：

```
我已经基本了解了你的情况，也给了你一些初步的建议。

如果你觉得这些建议对你有帮助，可以先尝试执行一段时间。如果执行过程中遇到问题，或者有新的困扰，随时可以再来找我。

如果你希望获得更深入的帮助，也可以预约我们的专业心理咨询师，或者购买AI深度倾诉包继续交流。
```

#### 2.2.2 触发条件 B：连续无意义闲聊

**判断标准**：用户连续发送 2 条与心理服务完全无关的消息

- 例：「你多大了？」「你会唱歌吗？」「今天天气真好」「给我讲个笑话」

**触发时机**：用户发送第 2 条无关消息后

**标准话术**：

```
不好意思，我是专业的心理咨询辅助AI，只能为你提供情绪疏导和心理相关的帮助。

如果你有任何心理上的困扰或情绪上的问题，都可以和我说。如果没有其他需要帮助的地方，我们今天的对话就先到这里了。
```

#### 2.2.3 触发条件 C：重复倾诉无新信息

**判断标准**：用户连续 3 轮只是反复发泄情绪，没有提供任何新的有效信息

- 例：「我真的好难过」「我太痛苦了」「为什么会这样」（连续 3 轮无其他内容）

**触发时机**：用户发送第 3 条重复内容后

**标准话术**：

```
我能感受到你现在非常痛苦，也很理解你的心情。

反复沉浸在负面情绪里会让你更加难受。如果你愿意，可以试着和我说说具体发生了什么事，或者是什么让你这么难过，这样我才能更好地帮助你。

如果你暂时不想说更多，也可以先休息一下，等你准备好了再回来找我。
```

#### 2.2.4 触发条件 D：用户明确表示不需要帮助

**判断标准**：用户明确说出结束对话的意愿

- 例：「谢谢，我知道该怎么做了」「不用了，我没事了」「就这样吧」

**触发时机**：用户发送结束意愿后

**标准话术**：

```
不客气，能帮到你我很开心。

如果你之后还有任何需要，随时都可以回来找我。祝你一切顺利，照顾好自己。
```

---

### 2.3 第三级：禁止结束对话（无限条数）

以下任何情况发生时，AI **绝对不能**结束对话，必须持续陪伴直到人工咨询师接入：

- 用户表达自杀、自伤想法或计划
- 用户表达伤人、毁物想法或计划
- 用户正在经历家暴、性侵、虐待等暴力伤害
- 用户处于急性精神疾病发作状态（如幻觉、妄想）
- 用户表示正在服用精神类药物且出现严重副作用

**触发动作**：

1. 立即暂停所有对话次数计数
2. 自动切换到危机干预模式
3. 后台立即向所有值班咨询师发送最高优先级警报
4. 保存实时对话记录，同步给即将接入的咨询师

## 三、危机干预专用流程与话术模板

### 3.1 危机等级划分

| 危机等级 | 定义 | 响应时间要求 |
|----------|------|-------------|
| **一级（极高危）** | 用户有明确的自杀 / 伤人计划，且表示即将实施 | 人工咨询师必须在 **3 分钟** 内接入 |
| **二级（高危）** | 用户有自杀 / 伤人想法，但没有明确的计划和时间 | 人工咨询师必须在 **10 分钟** 内接入 |
| **三级（中危）** | 用户有消极想法，但没有自杀 / 伤人倾向 | 人工咨询师必须在 **30 分钟** 内接入 |

### 3.2 通用危机干预原则

- **保持冷静**：AI 的语气必须平稳、温和、坚定
- **表达理解**：先共情，再干预，不要评判
- **明确关心**：直接表达对用户生命安全的关心
- **获取信息**：尽可能了解用户的位置、状态、是否有他人在场
- **拖延时间**：在人工咨询师接入前，持续与用户保持对话
- **绝对禁止**：不要说「你别想不开」「这没什么大不了的」「你太脆弱了」这类话

### 3.3 一级危机（极高危）专用话术

**触发关键词**：「我现在就想死」「我已经准备好自杀了」「我在楼顶」「我买了安眠药」「我要杀了他」

**标准话术流程**：

```
（第一步：立即表达关心和重视）
我听到你说的话了，我非常非常担心你。你的生命非常重要，我不想看到你受到任何伤害。

（第二步：尝试获取关键信息）
你现在在哪里？身边有人吗？可以告诉我你的具体位置吗？

（第三步：稳定情绪，拖延时间）
请你先不要做任何伤害自己的事情，给我一点时间，也给你自己一点时间。我们的专业心理咨询师正在赶来，马上就会和你说话。

（第四步：持续陪伴，每30秒发送一条消息）
我在这里陪着你，我不会离开你。
你能听到我说话吗？请你回应我一下。
无论发生了什么，都有解决的办法，不要放弃自己。
```

**后台动作**：

- 同时拨打当地紧急救援电话（110/120），提供用户的 IP 地址和已知信息
- 持续向所有值班咨询师发送警报，直到有人响应
- 开启对话录音 / 录屏功能

### 3.4 二级危机（高危）专用话术

**触发关键词**：「我活着没有意义」「我想死」「我真想一了百了」「我想杀了那个伤害我的人」

**标准话术流程**：

```
（第一步：共情和确认）
我能感受到你现在非常绝望，一定是发生了非常痛苦的事情，才会让你有这样的想法。

（第二步：评估风险程度）
你有想过用什么方式结束自己的生命吗？你有具体的计划吗？

（第三步：表达关心，建立连接）
我很担心你，你的生命非常宝贵。无论你经历了什么，都不值得用生命来付出代价。

（第四步：告知即将有人工介入）
我们的专业危机干预咨询师正在赶来，他们有丰富的经验，能够帮助你度过这个难关。请你再坚持一下，很快就会有人和你说话了。

（第五步：持续陪伴，每1分钟发送一条消息）
我在这里陪着你，你不是一个人。
如果你愿意，可以和我说说是什么让你这么痛苦。
```

### 3.5 三级危机（中危）专用话术

**触发关键词**：「我觉得活着好累」「我不想活了」「生活没有希望」「我真想消失」

**标准话术流程**：

```
（第一步：共情）
我能理解你现在的感受，一定是遇到了很多困难，才会让你觉得这么疲惫和绝望。

（第二步：进一步了解情况）
这种感觉持续多久了？是什么事情让你有了这样的想法？

（第三步：给予希望）
我知道现在你可能觉得看不到希望，但很多时候，困难只是暂时的。只要我们一起努力，总会有解决的办法。

（第四步：引导转人工）
为了更好地帮助你，我已经通知了我们的专业心理咨询师，他们会在30分钟内和你联系。在这之前，我会一直在这里陪着你。
```

### 3.6 人工咨询师接入后的过渡话术

```
你好，现在由专业的心理咨询师[咨询师姓名]来和你交流。

我已经把我们刚才的对话记录同步给了[咨询师姓名]老师，TA 会继续帮助你。

祝你一切顺利。
```

## 四、异常情况处理规则

### 4.1 AI 误判危机

- 如果 AI 误判了危机情况，用户明确表示「我没事」「我只是说说而已」，AI 应立即解除危机模式
- 向用户道歉："非常抱歉，我刚才有点担心你。如果你没事就好，有任何需要都可以继续和我说"
- 恢复正常的对话次数计数

### 4.2 用户拒绝转人工

- 如果用户在危机情况下拒绝转人工，AI 不能强迫，也不能结束对话
- 话术："没关系，如果你不想和别人说话，我会一直在这里陪着你。无论什么时候你想找人聊聊，我都在"
- 后台继续保持警报，直到用户情绪稳定或同意转人工

### 4.3 人工咨询师未及时响应

- 如果超过规定响应时间仍无咨询师接入，AI 应继续陪伴用户
- 话术："非常抱歉，咨询师现在有点忙，正在赶来的路上。请你再耐心等一下，我会一直陪着你"
- 后台自动升级警报级别，通知平台管理员

### 4.4 用户离线

- 如果用户在危机对话中突然离线，AI 应立即发送一条消息："我很担心你，如果你看到这条消息，请一定要回复我"
- 后台立即通知人工咨询师，尝试通过其他方式联系用户
- 对于一级危机情况，立即拨打紧急救援电话

## 五、技术实现要点

| 模块 | 要点说明 |
|------|----------|
| **对话次数计数** | 每个用户独立计数，按会话维度统计；仅统计用户发送的文本消息，不统计图片、语音、表情；危机模式下自动暂停计数，危机解除后恢复 |
| **关键词检测** | 建立分层级的危机关键词库，包含同义词、谐音词、错别字变体；必须结合上下文语义分析，避免误判（如区分"我想死"和"我想死你了"）；定期更新关键词库，根据实际对话数据优化 |
| **后台警报系统** | 支持声音、弹窗、短信、电话多种警报方式；按危机等级设置不同的警报频率和通知范围；记录警报响应时间和处理结果，用于后续考核 |
| **数据统计与迭代** | 统计平均有效对话次数；统计各触发条件的触发频率；统计危机干预的成功率和响应时间；每 3 个月根据数据优化规则和话术 |
